# Money Manager Backend

**Description**
A backend service for managing personal finances (APIs, DB, etc).

**Tech Stack**
• Java  
• Spring Boot  
• Maven

**Setup**
1. Clone the repo  
2. Run with `./mvnw spring-boot:run`  
3. Access APIs on localhost

**Endpoints**
GET /api/transactions
POST /api/transactions
